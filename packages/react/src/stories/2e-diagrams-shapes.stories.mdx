# ShapeNode

The **ShapeNode** component is conditionally rendered as a div, button
or hyperlink. Typically, they are associated with an icon, a title
and other properties.

### Default
<Canvas style={{ background: '#fff' }}>
  <ShapeNode title={'Title'} renderIcon={<User />} />
</Canvas>

</Canvas>

### Rounded square

</Canvas>

# Themes
> This version relies on <b>Carbon v11</b>. If you're using Carbon v10, <a href="https://carbon-charts-0x.netlify.app" target="_blank" rel="noreferrer">see the legacy demo site</a>.

Four Carbon themes (white, g10, g90 and g100) are included in the main CSS bundle file with usage shown below.

The two examples for CSS and SCSS assume you are using a bundler like Vite.js that allows direct imports of CSS/SCSS.

```html
<div id="my-sample-chart"></div>
```

**CSS:**

```ts
import { SimpleBarChart } from '@carbon/charts'
import '@carbon/charts/styles.css' // styles.min.css is also provided

const chartHolder = document.getElementById("my-sample-chart")
new SimpleBarChart(chartHolder, {
  data,
  options: {
    ...myChartOptions,
    theme: "g100" // Pass theme in options
  }
})
```

**SCSS (dependencies `sass`, `@carbon/styles` and `@carbon/import-once` are required):**

```ts
import { SimpleBarChart } from '@carbon/charts'
import '@carbon/charts/scss/index.css'

const chartHolder = document.getElementById("my-sample-chart")
new SimpleBarChart(chartHolder, {
  data,
  options: {
    ...myChartOptions,
    theme: "g100" // Pass theme in options
  }
})
```

# Carbon Charts - Vue

The Carbon Charts Vue library provides a collection of reusable charting components
to build websites and user interfaces.

Adopting the library enables developers to use consistent markup,
styles, and behavior in prototype and production work.

[Vue wrappers - storybook](https://charts.carbondesignsystem.com/vue)

<br/>
## Install

**with yarn:**
```bash
yarn add @carbon/charts-vue d3 d3-cloud d3-sankey
```

**with npm:**
```bash
npm install --save @carbon/charts-vue d3 d3-cloud d3-sankey
```

<br/>
## Getting started

To start using the `StackedBarChart` component, try the example below:
<br/>

###### HelloWorld.vue
```ts
import Vue from 'vue'
import ChartsVue from '@carbon/charts-vue'
import '@carbon/charts-vue/styles.css'

Vue.use(ChartsVue)

export default {
  components: {},
  data() {
    return {
      data: [
        // refer to tabular data format tutorial
      ],
      options: {
        // refer to chart specific options
      }
    }
  },
  template:
    '<CcvStackedBarChart :data="data" :options="options"></CcvStackedBarChart>',
}
```

#### Using the server
We recommend the use of
[Vue Storybook](https://github.com/storybookjs/storybook/tree/next/app/charts-vue) for developing components.

**Start the server:**
```bash
cd packages/vue
yarn storybook
```

Open browser to `http://localhost:6006/`.
